llvm::GlobalVariable * CreateMetadataVar (Twine Name, ConstantStructBuilder & Init, StringRef Section, CharUnits Align, bool AddToUsed)

CreateMetadataVar - Create a global variable with internal

linkage for use by the Objective-C runtime.

This is a convenience wrapper which not only creates the

variable, but also sets the section and alignment and adds the

global to the "llvm.used" list.

Parameters

Name - The variable name.
Init - The variable initializer; this is also used to define the type of the variable.
Section - The section the variable should go into, or empty.
Align - The alignment for the variable, or 0.
AddToUsed - Whether the variable should be added to "llvm.used".

void MemCmpExpansion (CallInst * CI, uint64_t Size, const TargetTransformInfo::MemCmpExpansionOptions & Options, const bool IsUsedForZeroCmp, const DataLayout & TheDataLayout, DomTreeUpdater * DTU)

Initialize the basic block structure required for expansion of memcmp call

with given maximum load size and memcmp size parameter.

This structure includes:

1. A list of load compare blocks - LoadCmpBlocks.

2. An EndBlock, split from original instruction point, which is the block to

return from.

3. ResultBlock, block to branch to for early exit when a

LoadCmpBlock finds a difference.

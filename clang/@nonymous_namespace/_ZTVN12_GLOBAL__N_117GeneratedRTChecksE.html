void create (Loop * L, const LoopAccessInfo & LAI, const SCEVPredicate & UnionPred, ElementCount VF, unsigned int IC, OptimizationRemarkEmitter & ORE)

Generate runtime checks in SCEVCheckBlock and MemCheckBlock, so we can

accurately estimate the cost of the runtime checks. The blocks are

un-linked from the IR and are added back during vector code generation. If

there is no vector code generation, the check blocks are removed

completely.

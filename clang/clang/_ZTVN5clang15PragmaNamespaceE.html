class PragmaNamespace

Defined at line 96 of file clang/include/clang/Lex/Pragma.h

PragmaNamespace - This PragmaHandler subdivides the namespace of pragmas,

allowing hierarchical pragmas to be defined. Common examples of namespaces

are "\#pragma GCC", "\#pragma STDC", and "\#pragma omp", but any namespaces

may be (potentially recursively) defined.

Public Methods

PragmaHandler * FindHandler (StringRef Name, bool IgnoreNull)

FindHandler - Check to see if there is already a handler for the

specified name. If not, return the handler for the null identifier if it

exists, otherwise return null. If IgnoreNull is true (the default) then

the null handler isn't returned on failure to match.

class OMPLoopRangeClause

Defined at line 1153 of file clang/include/clang/AST/OpenMPClause.h

This class represents the 'looprange' clause in the

'#pragma omp fuse' directive

Code

                                    
                                         {c}
                                         #pragma omp fuse looprange(1,2)
                                         {
                                           for(int i = 0; i < 64; ++i)
                                           for(int j = 0; j < 256; j+=2)
                                           for(int k = 127; k >= 0; --k)

class VerifyDiagnosticConsumer

Defined at line 38 of file clang/include/clang/Frontend/VerifyDiagnosticConsumer.h

VerifyDiagnosticConsumer - Create a diagnostic client which will use

markers in the input source to check that all the emitted diagnostics match

those expected. See clang/docs/InternalsManual.rst for details about how to

write tests to verify diagnostics.

void VerifyDiagnosticConsumer (DiagnosticsEngine & Diags)

Create a new verifying diagnostic client, which will issue errors to

the currently-attached diagnostic client when a diagnostic does not match

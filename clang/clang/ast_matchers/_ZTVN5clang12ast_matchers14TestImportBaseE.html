void testImportSequence (const CodeFiles & CodeSamples, const std::vector<ImportAction> & ImportActions, StringRef FileForFinalCheck, BindableMatcher FinalSelectPredicate, BindableMatcher VerificationMatcher)

Test an arbitrary sequence of imports for a set of given in-memory files.

The verification is done by running VerificationMatcher against a

specified AST node inside of one of given files.

Parameters

CodeSamples Map whose key is the file name and the value is the file content.
ImportActions Sequence of imports. Each import in sequence specifies "from file" and "to file" and a matcher that is used for searching a declaration for import in "from file".
FileForFinalCheck Name of virtual file for which the final check is applied.
FinalSelectPredicate Matcher that specifies the AST node in the FileForFinalCheck for which the verification will be done.
VerificationMatcher Matcher that will be used for verification after all imports in sequence are done.

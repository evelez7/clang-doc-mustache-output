  •  translateSourceRange (const SourceManager & SM, const LangOptions & LangOpts, const CharSourceRange & R)

    Translate a Clang source range into a CIndex source range.

    Clang internally represents ranges where the end location points to the

    start of the token at the end. However, for external clients it is more

    useful to have a CXSourceRange be a proper half-open interval. This routine

    does the appropriate translation.

    Defined at line 152 of file clang/tools/libclang/CIndex.cpp

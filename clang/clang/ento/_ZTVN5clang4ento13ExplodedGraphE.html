std::unique_ptr<ExplodedGraph> trim (ArrayRef<const NodeTy *> Nodes, InterExplodedGraphMap * ForwardMap, InterExplodedGraphMap * InverseMap)

Creates a trimmed version of the graph that only contains paths leading

to the given nodes.

Parameters

Nodes The nodes which must appear in the final graph. Presumably these are end-of-path nodes (i.e. they have no successors).
ForwardMap [out] An optional map from nodes in this graph to nodes in the returned graph.
InverseMap [out] An optional map from nodes in the returned graph to nodes in this graph.

setTableFields - Uses the opcode type, instruction context, opcode, and a

ModRMFilter as criteria to set a particular set of entries in the

decode tables to point to a specific uid.

Parameters

uid - The unique ID of the instruction.
is32bit - Instructon is only 32-bit
noPrefix - Instruction record has no prefix.
ignoresVEX_L - Instruction ignores VEX.L
ignoresVEX_W - Instruction ignores VEX.W
AddrSize - Instructions address size 16/32/64. 0 is unspecified

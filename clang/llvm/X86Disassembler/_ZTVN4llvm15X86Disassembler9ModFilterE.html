ModFilter - Filters based on the mod bits [bits 7-6] of the ModR/M byte.

Some instructions are classified based on whether they are 11 or anything

else. This filter performs that classification.

Public Methods

void ModFilter (bool r)

Constructor

Parameters

r True if the mod bits of the ModR/M byte must be 11; false otherwise. The name r derives from the fact that the mod bits indicate whether the R/M bits [bits 2-0] signify a register or a memory operand.

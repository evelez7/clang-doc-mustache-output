class AllocatorList

Defined at line 33 of file llvm/include/llvm/ADT/AllocatorList.h

A linked-list with a custom, local allocator.

Expose a std::list-like interface that owns and uses a custom LLVM-style

allocator (e.g., BumpPtrAllocator), leveraging

for the

implementation details.

Because this list owns the allocator, calling

with a different

list isn't generally safe. As such,

has been left out of the

interface entirely.

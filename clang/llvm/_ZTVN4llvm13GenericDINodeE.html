class GenericDINode

Defined at line 287 of file llvm/include/llvm/IR/DebugInfoMetadata.h

Generic tagged DWARF-like metadata node.

An un-specialized DWARF-like metadata node. The first operand is a

(possibly empty) null-separated

header that contains arbitrary

fields. The remaining operands are

and are pointers

to other metadata.

Uses the SubclassData32 Metadata slot.

class ValueAsMetadata

Defined at line 458 of file llvm/include/llvm/IR/Metadata.h

Value wrapper in the Metadata hierarchy.

This is a custom value handle that allows other metadata to refer to

classes in the Value hierarchy.

Because of full uniquing support, each value is only wrapped by a single

object, so the lookup maps are far more efficient than

those using ValueHandleBase.

The options string of a boolean type has the grammar:

bool_options :: "" | "Y" | "y" | "D" | "d" | "T" | "t"

==================================

| C | Meaning |

==================================

| Y | YES / NO |

| y | yes / no |

| D / d | Integer 0 or 1 |

| T | TRUE / FALSE |

| t | true / false |

| (empty) | Equivalent to 't' |

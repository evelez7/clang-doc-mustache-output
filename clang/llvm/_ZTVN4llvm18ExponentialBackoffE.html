class ExponentialBackoff

A class to help implement exponential backoff.

Example usage:

Code

                                    
                                           ExponentialBackoff Backoff(10s);
                                           do {
                                             if (tryToDoSomething())
                                               return ItWorked;
                                           } while (Backoff.waitForNextAttempt());
                                           return Timeout;

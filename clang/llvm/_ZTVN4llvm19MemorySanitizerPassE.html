struct MemorySanitizerPass

Defined at line 43 of file llvm/include/llvm/Transforms/Instrumentation/MemorySanitizer.h

A module pass for msan instrumentation.

Instruments functions to detect unitialized reads. This function pass

inserts calls to runtime library functions. If the functions aren't declared

yet, the pass inserts the declarations. Otherwise the existing globals are

used.

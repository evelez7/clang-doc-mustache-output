class GenericUniformityAnalysisImpl

Defined at line 330 of file llvm/include/llvm/ADT/GenericUniformityImpl.h

Analysis that identifies uniform values in a data-parallel

execution.

This analysis propagates divergence in a data-parallel context

from sources of divergence to all users. It can be instantiated

for an IR that provides a suitable SSAContext.

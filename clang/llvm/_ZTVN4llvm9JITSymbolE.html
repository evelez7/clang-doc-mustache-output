void JITSymbol (GetAddressFtor GetAddress, JITSymbolFlags Flags)

Create a symbol for a definition that doesn't have a known address

yet.

This constructor allows a JIT layer to provide a reference to a symbol

definition without actually materializing the definition up front. The

user can materialize the definition at any time by calling the getAddress

method.

Parameters

GetAddress A functor to materialize a definition (fixing the address) on demand.

class MapVector

Defined at line 36 of file llvm/include/llvm/ADT/MapVector.h

This class implements a map that also provides access to all stored values

in a deterministic order. The values are kept in a SmallVector

<

*, 0> and the

mapping is done with DenseMap from Keys to indexes in that vector.

class InstructionOperand

Defined at line 268 of file llvm/utils/TableGen/Common/GlobalISel/Patterns.h

An operand for an InstructionPattern.

Operands are composed of three elements:

- (Optional) Value

- (Optional) Name

- (Optional) Type

Some examples:

(i32 0):$x -> V=int(0), Name='x', Type=i32

0:$x -> V=int(0), Name='x'

$x -> Name='x'

i32:$x -> Name='x', Type = i32

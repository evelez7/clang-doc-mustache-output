Expected<XCOFFTracebackTable> create (const uint8_t * Ptr, uint64_t & Size, bool Is64Bits)

Parse an XCOFF Traceback Table from

with

bytes.

Returns an XCOFFTracebackTable upon successful parsing, otherwise an

Error is returned.

Parameters

Ptr [in] A pointer that points just past the initial 4 bytes of zeros at the beginning of an XCOFF Traceback Table.
Size [in,out] A pointer that points to the length of the XCOFF Traceback Table. If the XCOFF Traceback Table is not parsed successfully or there are extra bytes that are not recognized, will be updated to be the size up to the end of the last successfully parsed field of the table.

class OptTable

Defined at line 54 of file llvm/include/llvm/Option/OptTable.h

Provide access to the Option info table.

The OptTable class provides a layer of indirection which allows Option

instance to be created lazily. In the common case, only a few options will

be needed at runtime; the OptTable class maintains enough information to

parse command lines without instantiating Options, while letting other

parts of the driver still use Option instances where convenient.

unsigned int findNearest (StringRef Option, std::string & NearestString, Visibility VisibilityMask, unsigned int MinimumLength, unsigned int MaximumDistance)

Find the OptTable option that most closely matches the given string.

Parameters

Option [in] - A string, such as "-stdlibs=l", that represents user input of an option that may not exist in the OptTable. Note that the string includes prefix dashes "-" as well as values "=l".
NearestString [out] - The nearest option string found in the OptTable.
VisibilityMask [in] - Only include options with any of these visibility flags set.
MinimumLength [in] - Don't find options shorter than this length. For example, a minimum length of 3 prevents "-x" from being considered near to "-S".
MaximumDistance [in] - Don't find options whose distance is greater than this value.

Returns

The edit distance of the nearest string found.

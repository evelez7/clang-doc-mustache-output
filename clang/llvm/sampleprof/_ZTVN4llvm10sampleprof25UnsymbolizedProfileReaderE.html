class UnsymbolizedProfileReader

Defined at line 738 of file llvm/include/../tools/llvm-profgen/PerfReader.h

Format of unsymbolized profile:

[frame1

@

frame2

@

...] # If it's a CS profile

number of entries in RangeCounter

from_1-to_1:count_1

from_2-to_2:count_2

......

from_n-to_n:count_n

number of entries in BranchCounter

src_1->dst_1:count_1

src_2->dst_2:count_2

......

src_n->dst_n:count_n

[frame1

@

frame2

@

...] # Next context

......

Note that non-CS profile doesn't have the empty `[]` context.

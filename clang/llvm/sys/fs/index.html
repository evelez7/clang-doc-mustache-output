  • error_code createUniqueFile (const Twine & Model, int & ResultFD, SmallVectorImpl<char> & ResultPath, OpenFlags Flags, unsigned int Mode)

    Create a uniquely named file.

    Generates a unique path suitable for a temporary file and then opens it as a

    file. The name is based on

    with '%' replaced by a random char in

    [0-9a-f]. If

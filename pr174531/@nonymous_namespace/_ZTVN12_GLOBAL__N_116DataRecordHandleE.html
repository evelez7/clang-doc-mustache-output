DataStore record data: 4B + size? + refs? + data + 0

- 4-bytes: Header

- {0,4,8}-bytes: DataSize (may be packed in Header)

- {0,4,8}-bytes: NumRefs (may be packed in Header)

- NumRefs*{4,8}-bytes: Refs[] (end-ptr is 8-byte aligned)

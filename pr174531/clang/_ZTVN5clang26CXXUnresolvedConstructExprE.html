class CXXUnresolvedConstructExpr

Defined at line 3742 of file clang/include/clang/AST/ExprCXX.h

Describes an explicit type conversion that uses functional

notion but could not be resolved because one or more arguments are

type-dependent.

The explicit type conversions expressed by

CXXUnresolvedConstructExpr have the form

T(a1, a2, ..., aN)

,

where

is some type and

...,

are values, and

either

is a dependent type or one or more of the

a

's is

type-dependent. For example, this would occur in a template such

as:

When the returned expression is instantiated, it may resolve to a

constructor call, conversion function call, or some kind of type

conversion.

Code

                                    
                                           template<typename T, typename A1>
                                           inline T make_a(const A1& a1) {
                                             return T(a1);
                                           }

class OpaquePtr

Defined at line 51 of file clang/include/clang/Sema/Ownership.h

Wrapper for void* pointer.

This is a very simple POD type that wraps a pointer that the Parser

doesn't know about but that Sema or another client does. The PtrTy

template argument is used to make sure that "Decl" pointers are not

compatible with "Type" pointers for example.

Template Parameters

PtrTy Either a pointer type like 'T*' or a type that behaves like a pointer.
